## 图片风格建议

- **色彩**：使用明亮、柔和的色彩，符合二次元美学
- **构图**：简洁明了，突出主题
- **风格**：现代简约，避免过于复杂的元素
- **格式**：JPG 或 PNG 格式
- **大小**：建议每张图片不超过 500KB，确保加载速度

## 临时占位符

如果暂时没有图片，可以使用以下在线占位图服务：
- https://via.placeholder.com/750x400
- https://picsum.photos/750/400

只需将占位图 URL 替换到 index.html 中的 src 属性即可。
